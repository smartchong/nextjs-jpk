# 实战精品课

基于 Next.js 的精品课电商 wap 网站


## 开发

### 开发环境准备

- Node > 10
- global yarn

```bash
# 首次安装
yarn install
# 开发
yarn dev
```

浏览器打开 [http://localhost:3000](http://localhost:3000)

## 技术栈

- Next.js 9.4
- React 16
